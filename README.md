# Christmas tree animation: White lights, like on a string

A websocket server serving n seconds of simple lights!


The following environment variables can be used for configuration: 

For Consul:
- CONSUL_ENABLED (defaults to true)
- CONSUL_HOST
- CONSUL_PORT

For Zipkin:
- ZIPKIN_ENABLED (defaults to true)
- ZIPKIN_HOST
- ZIPKIN_PORT
- ZIPKIN_PROBABILITY

For the leds itself: 
- LED_STRIPS
- LEDS_PER_STRIP
